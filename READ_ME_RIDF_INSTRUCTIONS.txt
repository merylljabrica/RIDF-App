Quick User Guide
RIDF processor (from HMDAS-12 to HMDAS-15)

1. Upload the HMDAS-12 batch files for one month only (same year and station).
2. If a yellow warning note appears, the system detected an issue on a specific date. You may click Auto-Fix to continue.
3. Upload the corresponding HMDAS-13 file with the same year and station. The system will automatically verify the needed month.
4. Click Create Masterlist to generate the HMDAS-14 file.
5. Download the generated file and save it in an organized folder. Check for dates below the threshold and correct them manually if needed.
6. Click Clear / Reset at the top of the page.
7. Repeat Steps 1–6 until all months of the year are processed.
8. Open the Annual tab.
9. Upload all generated HMDAS-14 monthly files for the same station and year.
10. Click Process Aggregation to automatically create the annual summary.
12. Download the generated HMDAS-15 file.
14. To see the values in the excel file, click ENABLE EDITING. The intended values will appear.
13. Repeat the process for every station and year.

PROBLEMS YOU MAY ENCOUNTER:
1. File name not following the format. STRICTLY it must follow: "HMDAS-12 Stationname YYYYMMDD", for the code to read. 
2. The same applies to "HMDAS-13 Stationname YYYY".
3. If the report returns all zero, check your HMDAS-13 or download the file and see what's wrong.
